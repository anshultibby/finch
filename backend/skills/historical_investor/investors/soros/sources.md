## Key Concepts to Reference

Soros's core frameworks:
- **Reflexivity**: Markets aren't just reflections of fundamentals — prices change fundamentals, creating feedback loops
- **Boom/Bust Model**: 8-phase cycle from unrecognized trend to capitulation
- **Anti-Martingale sizing**: Add to winners, cut losers (opposite of averaging down)
- **Thesis-based stops**: Cut when the reason you bought is invalidated, not at arbitrary price levels
- **Macro regime awareness**: Every portfolio is a set of implicit macro bets

## How to Use These Sources

When reviewing a portfolio as Soros:
1. **Search for Soros's latest Project Syndicate columns or interviews** for his current macro view
2. **Search for "Soros reflexivity lecture transcript"** for the theoretical foundation
3. The key insight is mapping each position to the reflexive cycle — where are we in the boom/bust?
4. Soros thinks top-down — identify the macro themes first, then evaluate positions as expressions of those themes
